# Coding Club Portal Frontend

## Tech Stack

- [React](https://react.dev/) (with [Vite](https://vite.dev/) for building)
- [TailwindCSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- Custom UI Components

## Setting Up

```bash
# setting up frontend
cd client
npm install
npm run dev
```
